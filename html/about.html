iPXE web сборка

Зачем

Web интерфейс с подготовленными образами. Многие пользователи предпочитают скачать подготовленные образы
iPXE вместо того, чтоб собирать их из исходного кода.

Исходный код

Исходный код доступен на Github.
Комментарии, вопросы, предложения и правки приветствуются!
Пожалуйста, сделайте fork и отправьте pull request или Вы можете создать issue.
